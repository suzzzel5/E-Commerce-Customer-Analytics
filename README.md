E-Commerce Customer Analytics & Revenue Insights

--Project Overview--

This project demonstrates end-to-end data analysis skills including:
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Revenue and sales analysis
- Customer behavior segmentation
- Correlation analysis
- Business insights generation

**Goal:** Transform raw e-commerce data into actionable business intelligence.

Dataset

The dataset includes customer transaction data with features:
- Customer demographics
- Purchase history
- Product categories
- Membership information
- Satisfaction scores
- Spending patterns

Technologies Used

- **Python 3.9+** - Core programming language
- **Pandas** - Data manipulation and analysis
- **NumPy** - Numerical computations
- **Matplotlib & Seaborn** - Data visualization
- **Jupyter Notebook** - Interactive analysis
